# LotteryBangBangDa (彩票帮帮忙)

### The Most Realistic 3D Lottery Number Picker on iOS
### iOS 平台上最真实的 3D 彩票选号助手

**LotteryBangBangDa** utilizes Apple's SpriteKit physics engine to simulate real-world lottery machines. Say goodbye to boring random number generators!

**彩票帮帮忙** 利用 Apple SpriteKit 物理引擎模拟真实的彩票摇奖机。告别枯燥的随机数生成器，享受沉浸式的摇号体验！

---

## Features / 主要功能

* **Real 3D Physics:** Balls collide, bounce, and roll naturally.
    **真实物理：** 球体自然碰撞、反弹、滚动，拒绝伪随机。
* **Multiple Modes:** Supports "Big Mixer" and "Slot Machine" styles.
    **多种模式：** 支持“大搅拌机”和“老虎机”两种玩法。
* **Major Lotteries:** Double Color Ball, Super Lotto, 3D, and more.
    **主流彩种：** 支持双色球、大乐透、福彩3D、排列三/五。
* **History Management:** Save and copy your lucky numbers easily.
    **历史管理：** 轻松保存和复制您的幸运号码。
